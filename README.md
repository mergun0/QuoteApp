# Satır Arası / QuoteApp

Satır Arası is a Java + XML Android application for sharing and discovering quotes from movies, TV series and books.

## Current architecture

- Java + XML
- MVVM + Repository Pattern
- Firebase Authentication
- Cloud Firestore
- Firebase App Check
- Local Node.js admin panel for temporary v1.0 moderation/account-deletion operations

## Legal and release documents

v1.0 legal, privacy, KVKK, community-safety and account-deletion drafts are kept in:

```text
docs/legal/
```

These files are review-ready drafts, not final legal approval. Developer identity, contact information, public URLs, retention periods, Play Console declarations and legal review must be completed before production publication.

No website deployment, Play Console submission, Firestore Rules deployment or production data migration is implied by these documents.
