# Feature Backlog

## Quote publishing length and Plus ideas

- Evaluate a soft/hard publishing character limit after alpha usability testing.
- Consider a future premium/Plus long-form quote or extended note capability only after the core free experience is stable.
- Do not enforce quote length limits in the current alpha sprint.

## App-controlled theme selector

- Add an in-app theme selector after the dark-first alpha UI is stable.
- Keep current release dark-first and avoid device-theme-driven unreadable combinations.
- Evaluate Light / Dark / System options later without changing existing social or quote logic.
