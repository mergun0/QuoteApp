## QuoteApp Roadmap

QuoteApp is evolving from a personal quote app into a social quote platform for movie, series, and book lovers.

---

## ✅ v0.4.0-alpha — Current Foundation

Completed:

* Firebase Authentication
* Login / Register / Forgot Password
* Splash session control
* Quote CRUD
* Home feed
* Discover feed
* Public user profiles
* Quote detail screen
* Like system
* Favorites system
* Report system
* Search and filtering
* Spoiler support
* Share action
* MVVM architecture
* Repository pattern
* CODEX_RULES.md
* Achievement and level system design
